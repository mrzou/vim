### 把.vimrc放到用户目录下

-----------------------------------------------
### 安装vundle
1. 创建.vim/bundle目录
2. 切换到该目录下`cd ~/.vim/bundle`
3. `git clone https://github.com/gmarik/vundle.git` 
4. 开启一个vim窗口，执行`:BundleInstall` 安装插件

-----------------------------------------------
### 安装插件
1. 在.vimrc中添加Bundle "插件"
2. 然后打开一个窗口执行命令`:BundleInstall`

### 安装ctags
